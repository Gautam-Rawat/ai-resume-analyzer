# AI Resume Analyzer

An AI-based web application that analyzes resumes, calculates ATS scores, and provides personalized improvement suggestions.

## 🚀 Features
- Upload resumes (PDF/DOCX)
- Extract and analyze text using NLP
- ATS score calculation
- Keyword matching with job descriptions
- AI-generated improvement suggestions

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- AI/NLP: Python, OpenAI API
- Parsing: PDF/DOCX parser

## 📂 Project Structure
frontend/ → User interface  
backend/ → APIs  
ai-module/ → NLP & scoring logic  
data/ → Job descriptions  

## ⚙️ Setup Instructions

### Clone repo
git clone https://github.com/Gautam-Rawat/ai-resume-analyzer

### Install dependencies
cd frontend && npm install  
cd ../backend && npm install  

### Run project
cd backend && npm start  
cd ../frontend && npm start
